# 🚀 Daily Research Agent

An AI-powered research assistant built using **ASP.NET Core + OpenAI**.

👉 Provide a topic  
👉 The agent will:
- 🔍 Search for relevant information  
- 🧠 Generate a concise summary  
- 💾 Store the result  
- 📧 (Optional) Send via email
